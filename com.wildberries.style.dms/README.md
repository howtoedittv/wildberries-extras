# wildberries-extras/com.wildberries.style.dms



**wildberries-extras** is a collection of wildberries configs that currently don't exist on the official wildberries site.
this is the dms one

## install instructions
mv ~/wildberries-extras/com.wildberries.style.dms/ ~/.config/DankMaterialShell/themes/wildberries

go to the settings -> themes -> custom themes and choose the .json file in the directory that is presented in the upper instruction
