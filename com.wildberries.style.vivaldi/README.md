# wildberries-extras/com.wildberries.style.vivaldi



**wildberries-extras** is a collection of wildberries configs that currently don't exist on the official wildberries site.
this is the vivaldi one

## install instructions
go to vivaldi -> settings -> themes -> import themes

and link the zip file in this folder

i also recomend setting the wallpaper in this directory as your browser background
