# wildberries-extras


**wildberries-extras** is a collection of wildberries configs that currently don't exist on the official wildberries site.

---

## Getting Started

### Clone the repository:
```sh
git clone --depth 1 https://github.com/howtoedittv/wildberries-extras
cd wildberries-extras
```


## Features

- Dms config
- foot config
- go inside of the config folders for instructions on how to install the configs
