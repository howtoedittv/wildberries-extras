# wildberries-extras/com.wildberries.style.foot



**wildberries-extras** is a collection of wildberries configs that currently don't exist on the official wildberries site.
this is the foot one

## install instructions

sudo mv ~/wildberries-extras/com.wildberries.style.foot/maplemono/ /usr/share/fonts/

fc-cache -v 

mv ~/wildberries-extras/com.wildberries.style.foot/foot.ini ~/.config/foot/

# if it does not exist make it
mkdir ~/.config/foot

exit and reenter foot to apply changes
